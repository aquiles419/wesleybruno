Psicoterapia Personalizada

Ofereço psicoterapia personalizada, adaptada às necessidades únicas de cada cliente. A abordagem principal é a Terapia Cognitivo Comportamental (TCC), uma ferramenta eficaz para abordar questões específicas e promover mudanças positivas.

Terapia de Autoconhecimento

Destaco a importância do autoconhecimento no processo terapêutico. A TCC, aliada ao desenvolvimento do autoconhecimento, é uma combinação poderosa para lidar com transtornos psicológicos. Compreender a si mesmo é fundamental para superar desafios emocionais.

Como a TCC aborda questões específicas?


Identificação de Padrões de Pensamento

A TCC foca na identificação de padrões de pensamento negativos ou distorcidos que contribuem para o sofrimento emocional.

Como a TCC aborda questões específicas?


Modificação de Cognições Disfuncionais

Trabalhamos juntos para modificar essas cognições, promovendo uma perspectiva mais saudável e equilibrada.

Como a TCC aborda questões específicas?


Desenvolvimento de Habilidades

A TCC inclui o desenvolvimento de habilidades práticas para lidar com emoções, gerenciar o estresse e promover a resiliência.

Processo Terapêutico

1. Avaliação Inicial: Iniciamos com uma avaliação para compreender suas necessidades e metas terapêuticas.
2. Definição de Objetivos: Juntos, estabelecemos metas realistas e mensuráveis para orientar o processo terapêutico.
3. Sessões Regulares: As sessões são conduzidas regularmente, proporcionando um espaço seguro para explorar pensamentos e emoções.
4. Estratégias Práticas:Integramos estratégias práticas da TCC para enfrentar desafios específicos e promover mudanças positivas.
